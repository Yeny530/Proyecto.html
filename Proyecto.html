CONCEPTOS

1.MODAL

Un modal es una ventana que aparece sobre el contenido principal y bloquea la interacción hasta que se cierre.

2.POP-UP

Un popup es una ventana emergente que puede aparecer en cualquier parte de la pantalla sin necesariamente interrumpir la interacción.

3.Single Sign On (SSO)

Es un sistema que permite a los usuarios iniciar sesión una sola vez y acceder a múltiples aplicaciones o servicios sin volver a autenticarse.

4.DATO

Un dato es la representación básica de un hecho, expresado en números, letras, imágenes o sonidos. Por sí solo no tiene significado, pero al procesarse y organizarse se convierte en información útil, fundamental en la tecnología y el desarrollo de software.

5.INFORMACIÓN

La información es el conjunto de datos organizados y procesados que adquieren significado y permiten comprender, analizar o tomar decisiones. En tecnología, la información es esencial para el funcionamiento de sistemas, aplicaciones y procesos digitales.

6.DPI

El DPI en un mouse indica la sensibilidad con la que el dispositivo detecta el movimiento. Cuanto mayor es el DPI, el cursor se mueve más rápido y con mayor precisión en la pantalla con menos desplazamiento físico del mouse, siendo ideal para tareas como diseño o videojuegos.

7.ESTÁNDARES PARA CÓDIGO QR

Son normas que definen la estructura, tamaño, corrección de errores y formato de los códigos QR para garantizar su correcta lectura, como el estándar ISO/IEC 18004.

8.KICK OFF

Es la reunión inicial de un proyecto en la que se presentan objetivos, roles, cronograma y estrategias para comenzar el trabajo.

9.STAKEHOLDER

Es cualquier persona, grupo u organización que tiene interés o se ve afectada por un proyecto o sus resultados.

10.VENTANA DE CONTEXTO IA

Es la cantidad de información que un modelo de inteligencia artificial puede analizar a la vez para entender y generar respuestas coherentes.

11.VPN (TunnelBear)

Es una red privada virtual que cifra la conexión y oculta la dirección IP para proteger la privacidad en línea; TunnelBear es un ejemplo de este tipo de servicio.

12.INGENIERÍA SOCIAL

Es la manipulación psicológica de personas para obtener información confidencial o acceso a sistemas mediante engaños.

13.MARCA PERSONAL

Imagen digital que una persona proyecta en el ámbito tecnológico, mostrando sus habilidades y conocimientos para destacar y generar oportunidades profesionales.

14.HUELLA DIGITAL

Es el rastro de información que una persona deja al interactuar en internet y con dispositivos tecnológicos. Incluye datos como publicaciones, búsquedas, ubicación y actividad en redes sociales, los cuales reflejan su comportamiento y presencia en el mundo digital.

15.PNL (Programación Neurolingüística)

Es un enfoque que estudia cómo el lenguaje, los pensamientos y los patrones de comportamiento afectan la forma en que una persona actúa y se comunica.

16.SINTAXIS

Es el conjunto de reglas que define cómo se organizan las palabras y símbolos en un lenguaje, ya sea natural o de programación.

17.METADATO

Es información que describe a otros datos, como el autor, la fecha de creación o el formato de un archivo.

18.IDENTACIÓN

Es el uso de espacios o tabulaciones al inicio de líneas de código para mejorar su organización y legibilidad.

19.IDE (Entorno de Desarrollo Integrado)

Es un software que reúne herramientas como editor de código, compilador y depurador para facilitar la programación.

20.LLM (Large Language Model)

Es un modelo de inteligencia artificial entrenado con grandes cantidades de texto para comprender y generar lenguaje natural.

21.GUI (Interfaz Gráfica de Usuario)

Es la parte visual de un software que permite la interacción mediante elementos gráficos como ventanas y botones.

22.SHADOW

En tecnología, se refiere a copias ocultas o no oficiales de datos, procesos o configuraciones utilizadas para pruebas o respaldo.

23.COMPILAR

Es convertir el código fuente en un programa ejecutable que pueda interpretar la computadora.

24.MOCKUP

Es una representación visual estática que muestra cómo se verá un diseño o producto antes de desarrollarlo.

25.SCRIPT

Es un conjunto de instrucciones escritas en un lenguaje de programación que se ejecutan automáticamente para realizar tareas específicas.

26.HEURÍSTICA

Es una técnica basada en la experiencia que ayuda a encontrar soluciones rápidas y efectivas a problemas.

27.AVG

Es un software de seguridad que protege dispositivos contra virus, malware y otras amenazas digitales.

28.STACK OVERFLOW

Es una comunidad en línea donde programadores comparten preguntas y respuestas sobre desarrollo y tecnología.

29.VISUAL STUDIO

Es un entorno de desarrollo de Microsoft para crear aplicaciones en múltiples lenguajes y plataformas.

30.VISUAL STUDIO CODE

Es un editor de código ligero y gratuito que ofrece funciones y extensiones para programar de forma sencilla.

31.ENDPOINT

Es un punto de acceso en una API donde se intercambian datos entre sistemas mediante solicitudes y respuestas.

32.FAST API

Es un framework de Python que permite crear APIs rápidas y eficientes con soporte para documentación automática.

33.PARTICIÓN DE DISCO DURO

Es la división lógica de un disco físico para organizar mejor los datos y sistemas de archivos.

34.PARÁMETROS

Son valores que se pasan a una función o proceso para que trabaje con datos específicos.

35.TECNOLOGÍA

Es el conjunto de conocimientos, herramientas y procesos que el ser humano utiliza para resolver problemas, facilitar tareas y mejorar la calidad de vida. En el ámbito digital, se relaciona con sistemas informáticos, software y dispositivos electrónicos.

36.COMUNICACIÓN

Proceso mediante el cual se transmite información entre dos o más partes, ya sean personas, máquinas o sistemas, asegurando que el mensaje sea claro y entendible.

37.DISPOSITIVO

Aparato electrónico que permite realizar tareas digitales, como computadoras, teléfonos móviles, tablets o servidores, siendo esenciales en el mundo tecnológico actual.

38.PROGRAMACIÓN

Proceso de diseñar y escribir instrucciones que una computadora puede entender y ejecutar para realizar tareas específicas. Es fundamental para crear software, aplicaciones y sistemas.

39.ALGORITMO

Serie ordenada de pasos lógicos que permiten resolver un problema o realizar una tarea. Es la base del desarrollo de programas y aplicaciones.

40.CÓDIGO

Conjunto de instrucciones escritas en un lenguaje de programación que la computadora interpreta para ejecutar acciones. Representa la lógica detrás de un programa.

41.APLICACIÓN

Programa diseñado para cumplir una función específica, como una app móvil o web, creado para facilitar tareas y mejorar la experiencia de los usuarios.

42.SCRUM

Metodología ágil que organiza el trabajo en equipo en ciclos cortos llamados sprints, permitiendo mejorar la eficiencia y adaptarse a cambios en el desarrollo de software.

43.KANBAN

Sistema visual para organizar tareas y flujos de trabajo mediante tableros y tarjetas, facilitando la gestión de proyectos y la priorización de actividades.

44.MVP (Producto Mínimo Viable)

Versión básica de un producto que se lanza para probar su funcionalidad con usuarios y obtener retroalimentación antes de desarrollarlo completamente.

45.CIBERSEGURIDAD

Conjunto de técnicas y herramientas destinadas a proteger sistemas informáticos, redes y datos contra accesos no autorizados, ataques o robos de información.

46.ENCRIPTACIÓN

Proceso que convierte datos en un formato ilegible para protegerlos, asegurando que solo personas autorizadas puedan acceder a ellos.

47.FIREWALL

Barrera digital que controla el tráfico de información entre redes para proteger los sistemas contra amenazas externas y accesos no autorizados.

48.PHISHING

Técnica de fraude digital que busca engañar a las personas para obtener información personal, como contraseñas o datos bancarios, mediante correos o mensajes falsos.

49.BASE DE DATOS

Sistema organizado para almacenar y gestionar grandes cantidades de información de forma estructurada y accesible.

50.SQL

Lenguaje utilizado para crear, modificar y consultar datos en bases de datos relacionales, muy usado en desarrollo de software.

51.NoSQL

Tipo de base de datos que permite almacenar información no estructurada de forma flexible, ideal para grandes volúmenes de datos como redes sociales.

52.BACKUP

Copia de seguridad que resguarda la información importante, permitiendo su recuperación en caso de pérdida o fallo en los sistemas.

53.INTERNET

Red global que conecta millones de computadoras y dispositivos alrededor del mundo, permitiendo la comunicación, intercambio de información y acceso a servicios digitales.

54.LAN (Red de Área Local)

Tipo de red que conecta dispositivos dentro de un área limitada, como una casa, oficina o edificio, permitiendo compartir archivos e impresoras.

55.INTELIGENCIA ARTIFICIAL (IA)

Rama de la informática que busca crear sistemas capaces de imitar la inteligencia humana, aprendiendo, resolviendo problemas y tomando decisiones.

56.Wi-Fi

Tecnología inalámbrica que permite conectar dispositivos a internet sin necesidad de cables, siendo esencial para la conectividad en hogares y empresas.

57.MACHINE LEARNING

Subárea de la IA que permite que las máquinas aprendan automáticamente a partir de datos, mejorando su rendimiento con el tiempo sin ser programadas de forma explícita.

58.CHATBOT

Programa que utiliza IA para interactuar con usuarios mediante texto o voz, respondiendo preguntas y ofreciendo asistencia de forma automática.

59.AUTOMATIZACIÓN

Uso de tecnología para realizar tareas repetitivas sin intervención humana, aumentando la eficiencia y reduciendo errores en procesos digitales e industriales.

60.FRONTEND

Parte visual de una aplicación o sitio web con la que interactúa el usuario, diseñada para ofrecer una experiencia clara, atractiva y funcional.

61.API (Interfaz de Programación de Aplicaciones)

Conjunto de reglas que permite que diferentes programas se comuniquen e intercambien información de manera estructurada.

62.FRAMEWORK

Conjunto de herramientas y buenas prácticas que facilitan y agilizan el desarrollo de software, proporcionando una base sólida para crear aplicaciones.

63.SPRINT

Periodo corto de trabajo en un proyecto ágil donde el equipo se enfoca en completar tareas específicas y entregar avances funcionales.

64.BACKLOG

Lista priorizada de tareas, requisitos o mejoras pendientes por desarrollar en un proyecto, utilizada como guía para el equipo.

65.CONTRASEÑA

Clave secreta utilizada para proteger cuentas y sistemas, garantizando que solo usuarios autorizados puedan acceder.

66.AUTENTICACIÓN

Proceso que verifica la identidad de un usuario antes de permitirle el acceso a un sistema o servicio digital.

67.AUTORIZACIÓN

Determina qué acciones o recursos puede utilizar un usuario después de haberse autenticado correctamente.

68.MFA (Autenticación Multifactor)

Método de seguridad que requiere más de una forma de verificación, como contraseña y código enviado al teléfono, para mayor protección.

69.CLOUD COMPUTING

Tecnología que permite acceder a recursos informáticos como almacenamiento, servidores y software a través de internet sin depender de un solo equipo físico.

70.AWS (Amazon Web Services)

Plataforma de servicios en la nube que ofrece soluciones para almacenamiento, bases de datos, análisis y desarrollo de aplicaciones.

71.GOOGLE CLOUD

Servicio en la nube que proporciona herramientas para inteligencia artificial, big data, almacenamiento y desarrollo de software.

72.VIRTUALIZACIÓN

Proceso que crea versiones virtuales de recursos físicos como servidores, redes o sistemas operativos, optimizando el uso de hardware.

73.CONTROL DE CALIDAD

Proceso que asegura que un producto o software cumpla con los estándares y requisitos establecidos antes de ser lanzado.

74.PRUEBAS DE SOFTWARE

Etapa del desarrollo dedicada a identificar errores, fallos o problemas en el funcionamiento de una aplicación para garantizar su calidad.

75.DEPURACIÓN (Debugging)

Proceso de localizar y corregir errores en el código para asegurar que el programa funcione correctamente.

76.DOCUMENTACIÓN

Conjunto de manuales, guías y descripciones técnicas que explican el funcionamiento de un sistema o software.

77.E-COMMERCE

Venta y compra de productos o servicios a través de internet, utilizando plataformas y métodos de pago digitales.

78.PASARELA DE PAGO

Sistema que procesa pagos en línea de forma segura, como tarjetas de crédito, transferencias o criptomonedas.

79.BLOCKCHAIN

Tecnología que almacena información en bloques conectados entre sí, ofreciendo seguridad, transparencia y trazabilidad en transacciones digitales.

80.VIDEOJUEGOS

Programas de entretenimiento digital que combinan gráficos, sonido e interactividad, permitiendo a los usuarios participar en desafíos y experiencias virtuales. Pueden ser educativos, recreativos o competitivos.

81.UX (Experiencia de Usuario)

Proceso de diseñar productos digitales centrados en la satisfacción del usuario, buscando que la interacción sea sencilla, eficiente y agradable.

82.MOTOR DE VIDEOJUEGOS

Plataforma de desarrollo como Unity o Unreal Engine que proporciona herramientas para crear videojuegos, incluyendo física, gráficos, animaciones y sonido.

83.NPC

Personaje controlado por la computadora dentro de un videojuego que interactúa con el jugador o el entorno, aportando historia y desafíos.

84.GAMIFICACIÓN

Uso de elementos de los videojuegos, como puntos, niveles y recompensas, en otros contextos como educación o trabajo para motivar y mejorar la experiencia.

85.Google Chrome

es un navegador creado por la compañía Google INC. Considerado como el navegador más rápido del mundo.

86.PROTOTIPO

Versión inicial y simplificada de un producto que permite probar ideas y detectar mejoras antes de desarrollar la versión final.

87.ACCESIBILIDAD

Principio de diseño que asegura que un producto digital pueda ser utilizado por personas con distintas capacidades, incluyendo discapacidades físicas o sensoriales.

88.SENSOR

Dispositivo que detecta cambios en el entorno, como temperatura, luz o movimiento, y los convierte en datos que pueden ser procesados por un sistema.

89.REALIDAD AUMENTADA (AR)

Tecnología que combina elementos virtuales con el mundo real a través de dispositivos como celulares o gafas especiales, creando experiencias interactivas.

90.REALIDAD VIRTUAL (VR)

Entorno completamente digital que sumerge al usuario en una experiencia simulada, utilizada en videojuegos, educación y simuladores.

91.METAVERSO

Espacio virtual compartido donde las personas interactúan mediante avatares, combinando redes sociales, videojuegos y realidad virtual.

92.PROTECCIÓN DE DATOS

Conjunto de prácticas y normativas destinadas a resguardar la información personal para evitar su uso indebido o no autorizado.

93.PRIVACIDAD DIGITAL

Derecho de los usuarios a controlar qué información comparten en internet y cómo es utilizada por empresas y plataformas.

94.HACKING ÉTICO

Práctica realizada por expertos en ciberseguridad que buscan vulnerabilidades en sistemas para corregirlas y prevenir ataques malintencionados.

95.PLATAFORMA EDUCATIVA

Espacio virtual donde se centralizan herramientas y contenidos para la enseñanza y el aprendizaje, como Moodle, Blackboard o Coursera.

96.CERTIFICACIÓN DIGITAL

Reconocimiento oficial obtenido en línea que valida las habilidades y conocimientos adquiridos en un área específica.

97.TECNOLOGÍA VERDE

Desarrollo de soluciones digitales que reducen el impacto ambiental, como energías renovables, reciclaje electrónico y eficiencia energética.

98.SMART GRID (Red Inteligente)

Sistema de distribución eléctrica que utiliza sensores y tecnología digital para optimizar el consumo y la distribución de energía.

99.ECONOMÍA CIRCULAR

Modelo que busca reducir residuos al reutilizar y reciclar productos, integrando tecnología para la gestión eficiente de recursos.

100.MOVILIDAD SOSTENIBLE

Uso de medios de transporte tecnológicos y ecológicos como vehículos eléctricos y bicicletas compartidas para reducir la contaminación.

101.RED 5G

Tecnología de comunicación móvil que ofrece mayor velocidad, menor latencia y capacidad para conectar más dispositivos simultáneamente.

102.IoT (Internet de las Cosas)

Red de dispositivos físicos conectados a internet que recopilan y comparten datos para automatizar procesos y mejorar la eficiencia.

103.SENSOR DE PROXIMIDAD

Dispositivo que detecta la presencia de objetos cercanos sin contacto físico, utilizado en teléfonos, automóviles y sistemas de seguridad.

104.ROBÓTICA

Ciencia que diseña y crea robots para realizar tareas automáticas en la industria, medicina, educación y otros campos.

105.DRON

Aeronave no tripulada controlada de forma remota, utilizada para fotografía, seguridad, agricultura y transporte.

106.IMPRESIÓN 3D

Tecnología que fabrica objetos tridimensionales capa por capa a partir de un diseño digital, utilizada en medicina, arquitectura y manufactura.

107.BIG DATA

Conjunto de datos masivos que requieren herramientas avanzadas para ser almacenados, procesados y analizados de manera eficiente.

108.ANÁLISIS DE DATOS

Proceso de examinar datos para encontrar patrones, tendencias y obtener información útil para la toma de decisiones.

109.BI (Business Intelligence)

Conjunto de estrategias y herramientas que transforman datos en información valiosa para optimizar procesos empresariales.

110.DATA MINING

Técnica que utiliza algoritmos para descubrir patrones ocultos y relaciones en grandes volúmenes de datos.

111.CIENTÍFICO DE DATOS

Profesional encargado de recopilar, analizar e interpretar datos para generar conocimiento y soluciones estratégicas.

Proceso que extrae datos de diversas fuentes, los transforma y los carga en un sistema centralizado como un data warehouse.

113.DATA WAREHOUSE

Almacén centralizado que integra datos de distintas fuentes para análisis y generación de reportes empresariales.

114.DATA LAKE

Repositorio que almacena grandes volúmenes de datos en su formato original, estructurados y no estructurados.

115.KPI (Indicador Clave de Desempeño)

Métrica utilizada para evaluar el rendimiento de procesos, proyectos o estrategias dentro de una organización.

116.DASHBOARD

Panel visual que presenta métricas y datos clave en tiempo real para facilitar la toma de decisiones.

117.ERP (Enterprise Resource Planning)

Software que integra y gestiona procesos empresariales como finanzas, recursos humanos, producción y ventas.

118.CRM (Customer Relationship Management)

Sistema que gestiona las relaciones con clientes, optimizando la atención y fidelización.

119.CMS (Content Management System)

Plataforma que permite crear, gestionar y publicar contenido digital de forma sencilla, como WordPress o Joomla.

120.SEO (Search Engine Optimization)

Estrategias y técnicas para mejorar la visibilidad de un sitio web en motores de búsqueda como Google.

121.SEM (Search Engine Marketing)

Publicidad pagada en buscadores que busca aumentar la visibilidad y atraer tráfico a un sitio web.

122.PPC (Pago Por Clic)

Modelo de publicidad digital en el que el anunciante paga cada vez que un usuario hace clic en su anuncio.

Estrategia que utiliza personas influyentes en redes sociales para promocionar productos o servicios.

Campañas de comunicación y publicidad enviadas a través del correo electrónico para fidelizar clientes o captar nuevos.

125.AUTOMATIZACIÓN DE MARKETING

Uso de software para programar y ejecutar campañas de marketing de manera automática y personalizada.

126.CHAT COMERCIAL

Herramienta que permite la atención en línea y en tiempo real con clientes para resolver dudas y cerrar ventas.

127.Intel Core

Servicio que permite almacenar archivos en servidores en la nube, accesibles desde cualquier dispositivo con internet.

129.DRIVE

Plataforma de Google para almacenar, compartir y colaborar en documentos y archivos en la nube.

130.DROPBOX

Servicio de almacenamiento en la nube que facilita la sincronización y el intercambio de archivos.

131.ONEDRIVE

Servicio en la nube de Microsoft para guardar y compartir archivos en múltiples dispositivos.

132.MEGA

Plataforma de almacenamiento en la nube con opciones de cifrado para mayor privacidad de los datos.

133.ENCRIPTACIÓN DE PUNTA A PUNTA

Método de seguridad que cifra los datos desde su origen hasta su destino, asegurando que solo el emisor y el receptor puedan leerlos.

Computadora diseñada para gestionar grandes cantidades de datos y servicios en una empresa o red.

135.REDUNDANCIA

Práctica de duplicar componentes o procesos críticos para garantizar la continuidad en caso de fallos.

136.BACK-END

Parte del software que gestiona la lógica, base de datos y servidores, invisible para el usuario final.

137.SERVIDOR DEDICADO

Equipo completo destinado exclusivamente a alojar y ejecutar aplicaciones o servicios para una organización.

138.HOSTING

Servicio que ofrece espacio y recursos en servidores para alojar sitios web o aplicaciones.

139.DOMINIO

Nombre único que identifica a un sitio web en internet, como www.ejemplo.com.

Sistema que traduce nombres de dominio en direcciones IP para localizar sitios web en internet.

Identificador numérico único asignado a cada dispositivo conectado a una red.

Versión 4 del protocolo IP, utiliza direcciones de 32 bits para identificar dispositivos en la red.

Versión más reciente del protocolo IP que utiliza direcciones de 128 bits, ofreciendo mayor cantidad de combinaciones posibles.

144.PROTOCOLO HTTP

Reglas que permiten la transferencia de información en la web, base de la comunicación entre navegadores y servidores.

145.HTTPS

Versión segura de HTTP que cifra la información para protegerla durante la transmisión.

146.CERTIFICADO SSL

Archivo digital que autentica la identidad de un sitio web y habilita la navegación segura mediante HTTPS.

147.COOKIES

Archivos pequeños que un sitio web guarda en el navegador para recordar preferencias y datos de sesión.

148.CACHE

Memoria temporal que almacena datos para acelerar el acceso a información recurrente.

149.CDN (Content Delivery Network)

Red de servidores distribuidos que optimiza la entrega de contenido en línea, mejorando la velocidad y disponibilidad.

150.EDGE COMPUTING

Tecnología que procesa datos cerca de donde se generan para reducir la latencia y mejorar la eficiencia.

151.GPU (Unidad de Procesamiento Gráfico)

Procesador especializado en cálculos gráficos y de imágenes, esencial en videojuegos e inteligencia artificial.

152.CPU (Unidad Central de Procesamiento)

Componente principal de una computadora que interpreta y ejecuta instrucciones de programas.

153.RAM (Memoria de Acceso Aleatorio)

Memoria temporal que almacena datos y programas en uso para un acceso rápido por parte de la CPU.

154.ROM

Memoria de solo lectura que contiene instrucciones esenciales para el arranque de un dispositivo.

155.SSD

Unidad de almacenamiento que utiliza memoria flash para ofrecer mayor velocidad y rendimiento que los discos duros tradicionales.

156.HDD

Disco duro tradicional que almacena datos en discos magnéticos giratorios.

Servicio que permite jugar videojuegos en línea sin necesidad de hardware potente, ejecutando el juego en servidores remotos.

158.STREAMING

Transmisión de contenido multimedia en tiempo real a través de internet sin necesidad de descargarlo previamente.

159.PODCAST

Programa de audio digital disponible en plataformas en línea para ser escuchado bajo demanda.

160.VLOG

Contenido en formato de video donde una persona comparte experiencias, ideas o información en línea.

161.LIVE STREAM

Transmisión en vivo de eventos o actividades mediante plataformas digitales como YouTube o Twitch.

162.CLOUD BACKUP

Copia de seguridad de datos almacenada en servidores en la nube para recuperación en caso de pérdida.

163.CRIPTOMONEDA

Moneda digital descentralizada basada en tecnología blockchain, como Bitcoin o Ethereum.

164.WALLET DIGITAL

Aplicación que permite almacenar y gestionar criptomonedas de forma segura.

165.NFT (Token No Fungible)

Activo digital único que representa la propiedad de un contenido como arte, música o coleccionables en blockchain.

166.SMART CONTRACT

Programa almacenado en blockchain que se ejecuta automáticamente cuando se cumplen ciertas condiciones.

167.CIBERDELITO

Actividad ilegal realizada a través de medios digitales, como fraudes, robos de identidad o ataques informáticos.

168.SPAM

Mensajes no solicitados enviados de forma masiva, generalmente con fines publicitarios o fraudulentos.

169.MALWARE

Software malicioso diseñado para dañar, robar información o tomar control de sistemas.

170.RANSOMWARE

Tipo de malware que secuestra datos y exige un pago para liberarlos.

171.SPYWARE
